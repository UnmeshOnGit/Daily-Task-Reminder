# Daily Task Reminder

Daily Task Reminder is a simple and modern Android application built using Kotlin that helps users manage their daily tasks with customizable reminders and notifications. The app allows users to create tasks for today, tomorrow, or recurring daily activities and receive timely notifications to stay productive and organized.

---

## ✨ Features

* Add and manage daily tasks easily
* Create reminders with custom date and time
* Notifications for scheduled tasks
* Simple and clean one-page UI
* Organize tasks like:

  * Today
  * Tomorrow
  * Daily
* Edit and delete tasks
* Lightweight and easy to use
* Offline functionality

---

## 🛠️ Tech Stack

* **Language:** Kotlin
* **IDE:** Android Studio
* **UI:** XML / Jetpack Compose 
* **Database:** Room Database / SQLite
* **Notifications:** AlarmManager & NotificationManager
* **Architecture:** MVVM

## 🚀 Getting Started

### Prerequisites

Before running the project, make sure you have:

* Android Studio installed
* Android SDK configured
* Kotlin support enabled

---

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/UnmeshOnGit/Daily-Task-Reminder.git
```

2. Open the project in Android Studio

3. Sync Gradle files

4. Run the app on an emulator or physical device

---

## 🔔 Reminder Functionality

The app uses Android notification services to remind users about scheduled tasks at selected times. Notifications work locally on the device without requiring internet access.
